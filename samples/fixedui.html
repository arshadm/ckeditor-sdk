Fixed User Interface Documentation

Fixed user interface is the default UI for classic editor. The basic features of fixed UI are:

  • Toolbar is always visible.
  • The location of a toolbar and bottom bar is static and do not change.
  • The editing area takes a predefined amount of space, regardless of the size of the content inside.

The default behavior of an editor with fixed UI can be adopted though:

  • Toolbar position is configurable with config.toolbarLocation
  • The size of editing area can be automatically adjusted when fixed UI is used – check the Auto Grow feature.
  • The size of the editor can be manually controlled by using the resize handle at the bottom.

Apollo 11

Apollo 11 was the spaceflight that landed the first humans, Americans Neil Armstrong and Buzz Aldrin, on the Moon on July 20, 1969, at 20:18 UTC. Armstrong became the first to step onto the lunar surface 6 hours later on July 21 at 02:56 UTC.

Armstrong spent about three and a half two and a half hours outside the spacecraft, Aldrin slightly less; and together they collected 47.5 pounds (21.5 kg) of lunar material for return to Earth. A third member of the mission, Michael Collins, piloted the command spacecraft alone in lunar orbit until Armstrong and Aldrin returned to it for the trip back to Earth.

Note: Fixed UI for inline editor is provided by a dedicated, optional plugin: Div Editing Area.
Please be aware that this plugin has a couple of drawbacks:

  • Under the hood, the element replaced by CKEditor is hidden automatically and Div Based Editor is created next to it. Such behavior might affect the styles of content inside, e.g. because of the lack of original element ID attribute used by CSS selectors.
  • The Div Based Editor is rendered using multiple nested <div> elements. It might affect the styles of content inside as well.
  • Last but not least, the Auto Grow feature is not fully supported – config.autoGrow_minHeight is not respected.

For Fixed UI experience with inline editor consider using Shared Spaces instead, where these limitations do not exist,
