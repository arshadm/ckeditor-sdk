Creating Mathematical Formulas

CKEditor 4.3 introduced the Widget System that paved way to a handful of innovative widget plugins. One of these is the optional Mathematical Formulas plugin that provides the MathJax widget which can be used to create or modify equations using TeX.

When the Mathematical Formulas plugin is enabled, the   button is automatically added to the toolbar. Once clicked, it opens the Mathematics in TeX dialog window that lets you write an equation and see its preview.

A few configuration options are available to fine-tune this feature, including setting the default class for an element that will be converted into a mathematical formulas widget and changing the default path to the MathJax library. Refer to the Creating Mathematical Formulas article to learn more about this feature.

Please note that this plugin is not compatible with Internet Explorer 8.
