Inserting Media Embed objects

The Media Embed plugin allows you to insert videos, images, tweets, music and much more from popular services.

Simply enable the plugin and paste a link to the desired content. You can also use the Insert Media Embed   toolbar button to open a dialog where you can insert a link. 

CKEDITOR.replace( 'textarea_id', {
	extraPlugins: 'mediaembed'
} );

Media Embed paste support

By default Media Embed plugin comes with feature to detect embeddable content link in pasted text. In case of positive match it will automatically convert link into Media Embed widget.

This feature may be disabled by the CKEDITOR.config.mediaEmbed_disablePasteUpcast config property.

Output strategies

Some developers might actually need to customize widget behaviour output HTML.

Media embed allows you to customize output using different output strategies.

The plugin ships with two output strategies:

  • default - Includes whole code to the output HTML.
  • oembed - Places oembed element with media link as an inner text.

Output strategies may be configured using the CKEDITOR.config.mediaEmbed_output property. 

CKEDITOR.replace( 'editor3', {
	extraPlugins: 'mediaembed',
	mediaEmbed_output: 'oembed'
} );
