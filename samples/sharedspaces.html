Shared Toolbar and Bottom Bar

The optional Shared Space plugin makes it possible to share the same editor toolbar and bottom bar among several CKEditor instances.

For each editor instance you can define the elements that will be shared. In the example below both editor types (classic and inline) are combined. Please note that the second classic editor instance only uses the shared toolbar alongside its own bottom bar (with the elements path). This is possible thanks to the CKEDITOR.config.sharedSpaces configuration setting

Refer to the Shared Toolbar and Bottom Bar article to learn more about this feature.
