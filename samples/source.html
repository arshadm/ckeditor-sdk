Source Code Editing

The main CKEditor strength is WYSIWYG editing, but the underlying format of the editor content is HTML. However, working directly with HTML code is also possible thanks to the source code editing feature provided by the Source Editing Area and Source Dialog plugins.

Both plugins introduce the   toolbar button. Refer to the Source Code Editing article to learn more about this feature as well as the functional differences between the plugins.

The following editor instances show how both plugins can be combined with classic and inline editor. The first one is a classic editor instance with default settings, using the source editing area. The second one is an inline editor with the source dialog. The third instance is a classic editor using the source dialog instead of the default source editing area.

This inline editor instance is using the Source Dialog plugin for source editing.

Follow the steps below to try it out:

  • Click the Source button to display the HTML source of this text in the Source dialog window.
  • Close the Source dialog window to return to the WYSIWYG view.

The following configuration option was used: 

config.extraPlugins = 'sourcedialog';
