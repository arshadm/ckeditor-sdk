UI Color Picker

The editor UI color can also be changed dynamically by the user with help of the color picker. This tool is enabled by the UI Color Picker plugin that needs to be available in the editor build.

When the UI Color Picker plugin is enabled, the   button is automatically added to the toolbar. The tool can then be used by the developer to conveniently try out a few options and adjust the color of the editor user interface. The code that needs to be pasted to the editor configuration in order to preserve the color choice is displayed in the dialog window.

If this plugin is available to end-users, they will be able to use it to change the color of the editor interface to their liking for the duration of their visit.

Please note that this configuration option can only be used for editor skins that are compatible with the so-called "chameleon" feature, for example Moono and Kama.

The editor example below was customized by setting the following configuration option: 

config.extraPlugins = 'uicolor';

Refer to the Adjusting Editor UI Color article to learn more about this feature.
